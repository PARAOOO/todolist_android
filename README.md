## Description
TODOLIST는 나만의 Todolist를 만드는 장기 프로젝트입니다. 
기본적인 Todolist에서 조금씩 기능을 추가해 나가는 MVP 방식으로 개발됩니다.

## Tech Stack
- Kotlin
- Android, Compose
- Coroutine, Flow
- Koin
- Room
- MVI, Clean Architecture
